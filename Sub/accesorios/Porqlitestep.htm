Litestep otra forma de Utilizar tu maquina

El theme incluye todos los módulos, ejecutable, no necesita de ninguna DISTRO. Explica en el readme donde encontrar la información para Instalarlo y brevemente como hacerlo en Windows 9X.
Sino simplemente baja un Instalador y después ejecuta el Explorer borra todo el contenido de la carpeta y pone todos los archivos como están en ella, y ya esta.

Se incluye también:

• En formato de Photoshop el archivo para crear tus propios accesos directos
• Las tipografías para poder modificar tus menús
• Pero no se Incluye archivo modificable de los accesos directos, si le queres poner otro nombre a tu PC simplemente borra la Palabra y escribí la que quieras.

Todas las Imágenes son .BMP para facilitar la compatibilidad con todas las aplicaciones de litestep que aunque algunas utilizan otros sistemas de imágenes, algunas solo soportan .BMP
